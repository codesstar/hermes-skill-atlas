OpenClaw 迁移 Skill（57 个）

通过 hermes claw migrate 从 OpenClaw 迁移。主要是角色扮演型（CEO/CTO/CFO 等）和文本处理型（polish/distill/clarify 等）。功能可用但大多是 prompt 包装，没有真工具调用。

社区忠告

  1. 别装太多 — "先装 1 个记忆 skill + 1 个工具 skill，够了再加"
  2. skill-factory 生成的 skill 需要人工审核 — "它总觉得自己做得很好"
  3. 注意 token 消耗 — Hindsight + 贵模型一天能烧 $50+
  4. 外部目录的 skill 可能报错 — 关键 skill 放默认目录里更稳
